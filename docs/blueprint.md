# **App Name**: Auditgelap

## Core Features:

- Landing Page Exposition: Display a landing page with content structured as exposition, rising action, climax, falling action, and resolution to hook the user.
- AI-Powered Audit: Utilize Gemini API to generate a brutal diagnosis of the user's situation based on their input, outputting data in JSON format.
- Financial Loss Calculation: Calculate and display the financial cost of the user's delays and inaction in IDR based on AI assessment tool.
- Growth Loss Percentage: Calculate and display the percentage of growth lost due to the user's delays and inaction based on AI tool.
- Strategic Command Generation: Generate strategic commands based on the audit's findings using AI to provide actionable steps for the user.
- Data Visualization: Display the diagnosis, financial loss, and growth loss in charts and figures using data returned in JSON format.
- Call To Action: Implement a 'Start Audit Gratis' button.

## Style Guidelines:

- Background: Deep Black (#000000).
- Accent for Gains: Neon Green (#39FF14) for positive financial figures.
- Accent for Losses: Blood Red (#8B0000) for negative financial figures.
- Font: 'Source Code Pro' monospace font for an analytical and technical feel.
